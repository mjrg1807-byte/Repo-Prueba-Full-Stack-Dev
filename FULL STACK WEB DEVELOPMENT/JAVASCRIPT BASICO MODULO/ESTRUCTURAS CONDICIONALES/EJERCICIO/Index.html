Ejercicios Prácticos

Verificar si un número es positivo, negativo o cero

Crea un código que devuelva un mensaje indicando si un número es positivo, negativo o cero.

Edad legal para Votar

Crea un código que devuelva "Puede votar" si la edad es mayor o igual a 18, y "No puede votar" si es menor.

Mayor de dos números

Crea un código que devuelva el mayor de dos números ingresados. Si son iguales, devuelve un mensaje indicándolo.

Determinar si el número es divisible por 5

Crea un código que devuelva true si el número es divisible por 5, y false si no lo es.

Calificación de un estudiante

Crea un código que, dada una nota entre 0 y 10, devuelva:
- Nota >= 9: "Excelente"
- Nota >= 7 y < 9: "Buena"
- Nota >= 5 y < 7: "Regular"
- Nota < 5: "Insuficiente"
