DOM (Document Object Model)

El DOM es la forma en la que el navegador representa el HTML como una estructura de objetos. Permite a JavaScript interactuar con los elementos de una página web.

¿Qué es el DOM?

El DOM convierte el HTML en una estructura jerárquica que JavaScript puede leer y modificar. 

    document
    └── body
        └── p
            └── "Hola"

Objeto document

El objeto document representa toda la página web y es el punto de acceso al DOM. 

    document

Seleccionar elementos

Por ID

    document.getElementById("miId");

Con querySelector

    document.querySelector(".clase");
    document.querySelector("p");

Modificar contenido

    elemento.textContent = "Nuevo texto";

Modificar estilos

    elemento.style.color = "red";

Crear elementos

    let nuevo = document.createElement("p");
    nuevo.textContent = "Hola nuevo";

Eliminar elementos

    elemento.remove();

Eventos

Los eventos permiten ejecutar código cuando el usuario interactúa con la página. 

    elemento.addEventListener("click", function() {
    console.log("Click!");
    });

Funcionamiento general

  1. El usuario realiza una acción
  2. JavaScript detecta el evento
  3. Se ejecuta código
  4. El DOM se modifica

Ejemplo práctico

    let boton = document.querySelector("button");

    boton.addEventListener("click", function() {
    document.body.style.background = "black";
    });

Errores comunes

  • No conectar correctamente JavaScript con HTML
  • Seleccionar mal los elementos
  • Ejecutar código antes de que cargue el DOM

Resumen

  • DOM → representación del HTML como objetos
  • document → acceso principal al DOM
  • Permite seleccionar, modificar, crear y eliminar elementos
  • Eventos → interacción con el usuario
