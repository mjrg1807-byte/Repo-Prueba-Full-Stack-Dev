Objetos en JavaScript

Un objeto es una estructura que permite almacenar datos relacionados en forma de clave y valor. Se utiliza para representar entidades del mundo real como personas, productos o usuarios.

¿Qué es un objeto?

let persona = {
  nombre: "Mauro",
  edad: 25,
  activo: true
};

Cada dato dentro del objeto se denomina propiedad.

Propiedades

Un objeto está formado por pares clave → valor. 

nombre: "Mauro"
  • Clave → nombre
  • Valor → "Mauro"

Diferencia entre Array y Objeto

  • Array → lista ordenada de valores
  • Objeto → conjunto de datos con significado
// Array
let datos = ["Mauro", 25];

// Objeto
let persona = {
  nombre: "Mauro",
  edad: 25
};

Acceder a propiedades

Notación de punto

persona.nombre

Notación de corchetes

persona["edad"]

Modificar propiedades

persona.edad = 30;

Añadir propiedades

persona.ciudad = "Madrid";

Eliminar propiedades

delete persona.activo;

Objetos complejos

Un objeto puede contener arrays u otros objetos. 

let usuario = {
  nombre: "Mauro",
  hobbies: ["gym", "programar"],
  direccion: {
    ciudad: "Madrid",
    pais: "España"
  }
};

Métodos (funciones dentro de objetos)

let persona = {
  nombre: "Mauro",
  saludar: function() {
    console.log("Hola!");
  }
};

Una función dentro de un objeto se llama método.

Cuándo usar objetos

  • Cuando tienes datos relacionados
  • Para representar entidades (persona, producto, usuario)
  • Cuando necesitas organizar información de forma clara

Errores comunes

  • Confundir objetos con arrays
  • Olvidar comas entre propiedades
  • Acceder mal a propiedades (sin comillas en corchetes)

Resumen

  • Objeto → conjunto de datos con clave y valor
  • Se accede con punto o corchetes
  • Permite modificar, añadir y eliminar propiedades
  • Puede contener arrays, objetos y funciones
