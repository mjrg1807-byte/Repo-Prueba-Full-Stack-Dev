Introducción a JavaScript

JavaScript es un lenguaje de programación que permite añadir interactividad a una página web. Gracias a JavaScript, las webs dejan de ser estáticas y pasan a responder a las acciones del usuario.

¿Para qué sirve JavaScript?

  • Crear interactividad en la web
  • Responder a acciones del usuario
  • Modificar el contenido HTML
  • Cambiar estilos CSS dinámicamente

Relación con HTML y CSS

  • HTML → estructura
  • CSS → diseño
  • JavaScript → comportamiento

Variables

Las variables permiten guardar información para utilizarla más adelante. 

let nombre = "Mauro";
let edad = 25;
  • string → texto
  • number → números
  • boolean → verdadero o falso

Condicionales

Permiten tomar decisiones en función de una condición. 

if (edad > 18) {
  console.log("Eres mayor de edad");
}

Bucles

Permiten repetir un bloque de código varias veces. 

for (let i = 0; i < 5; i++) {
  console.log(i);
}

Funciones

Son bloques de código reutilizables que se pueden ejecutar cuando se necesiten. 

function saludar() {
  console.log("Hola!");
}

saludar();

DOM (Document Object Model)

El DOM permite a JavaScript interactuar con el HTML. 

<p id="texto">Hola</p>

document.getElementById("texto").textContent = "Adiós";

Permite modificar contenido, estilos y estructura de la página.

Eventos

Permiten ejecutar código cuando el usuario realiza una acción. 

let boton = document.querySelector("button");

boton.addEventListener("click", function() {
  console.log("Has hecho click");
});

Manipulación del DOM

  • Cambiar texto
  • Cambiar estilos
  • Crear elementos
  • Eliminar elementos
document.body.style.background = "black";

Errores comunes

  • No enlazar el archivo JavaScript
  • No entender el DOM
  • Errores de sintaxis
  • Intentar aprender todo de golpe

Consejo clave

Practica constantemente. Usa la consola del navegador (F12) para probar código, cometer errores y aprender de ellos.

Pseudocódigo y Diagramas de Flujo

El pseudocódigo y los diagramas de flujo son herramientas que permiten planificar y entender la lógica de un programa antes de escribir código real.

¿Qué es el pseudocódigo?

Es una forma de escribir la lógica de un programa utilizando lenguaje natural, sin necesidad de seguir la sintaxis de un lenguaje de programación.

Su objetivo es centrarse en la lógica y no en el código.

Ejemplo

INICIO

pedir edad

SI edad >= 18 ENTONCES
    mostrar "Eres mayor de edad"
SINO
    mostrar "Eres menor de edad"

FIN

¿Para qué sirve?

  • Organizar ideas antes de programar
  • Evitar errores
  • Entender problemas complejos
  • Planificar soluciones

¿Qué es un diagrama de flujo?

Es una representación visual de un proceso mediante símbolos y flechas.

Elementos básicos

  • Óvalo → Inicio / Fin
  • Rectángulo → Acción
  • Rombo → Decisión
  • Flechas → Dirección del flujo

Ejemplo

[Inicio]
   ↓
[Pedir edad]
   ↓
¿edad >= 18?
   ↓        ↓
Sí         No
↓           ↓
[Mayor]   [Menor]
   ↓
[Fin]

Diferencias

  • Pseudocódigo → texto
  • Diagrama de flujo → visual
  • Pseudocódigo → rápido de escribir
  • Diagrama → más fácil de entender

Cómo utilizarlos correctamente

  1. Entender el problema
  2. Escribir el pseudocódigo
  3. (Opcional) crear diagrama de flujo
  4. Convertirlo a código

Ejemplo práctico

Pseudocódigo

INICIO

para i desde 1 hasta 5
    mostrar i

FIN

JavaScript

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

Traducción a JavaScript

  • SI → if
  • SINO → else
  • PARA → for
  • MIENTRAS → while
  • MOSTRAR → console.log

Ejemplo real

Problema: Contador con botón

Pseudocódigo

INICIO

contador = 0

cuando el usuario haga click:
    contador = contador + 1
    mostrar contador

FIN

Errores comunes

  • Empezar a programar sin planificar
  • No definir bien el problema
  • Confundir lógica con sintaxis

Consejo clave

Si no sabes cómo programar algo, escríbelo primero en pseudocódigo. Esto te ayudará a entender la lógica antes de pasar a JavaScript.
