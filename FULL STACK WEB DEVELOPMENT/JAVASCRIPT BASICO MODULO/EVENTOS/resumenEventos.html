Eventos en JavaScript

¿Qué es un evento?

Un evento es una acción que ocurre en la página y que JavaScript puede detectar y gestionar.

  • Click en un botón
  • Escribir en un input
  • Pasar el ratón por un elemento
  • Cargar la página

Estructura básica

elemento.addEventListener("evento", function() {
  // código a ejecutar
});
  • elemento: elemento del DOM
  • evento: acción que se detecta
  • función: lo que ocurre cuando se activa

Ejemplo básico

<button id="btn">Click</button>

let boton = document.getElementById("btn");

boton.addEventListener("click", function() {
  console.log("Has hecho click");
});

Eventos más comunes

  • click → al hacer click
  • mouseover → al pasar el ratón
  • mouseout → al salir
  • input → al escribir
  • submit → en formularios

Ejemplo con input

input.addEventListener("input", function() {
  console.log(input.value);
});

Errores comunes

❌ Incorrecto

boton.addEventListener("click", saludar());

Esto ejecuta la función inmediatamente.

✅ Correcto

boton.addEventListener("click", saludar);

Funcionamiento

  1. El usuario realiza una acción
  2. Se dispara un evento
  3. JavaScript lo detecta
  4. Se ejecuta una función
  5. El DOM puede cambiar

Ejemplo práctico

let boton = document.querySelector("button");

boton.addEventListener("click", function() {
  document.body.style.background = "blue";
});

Resumen

  • Evento → acción del usuario
  • addEventListener → escucha eventos
  • Función → ejecuta el código
  • Permite crear interactividad en la web
