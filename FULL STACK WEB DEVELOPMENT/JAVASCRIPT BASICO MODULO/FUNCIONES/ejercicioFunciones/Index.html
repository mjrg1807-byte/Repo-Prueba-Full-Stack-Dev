Ejericio 1
Crear una funcion llamada saludar que acepte un nombre como argumento y retorne un mensaje de saludo como: "¡Hola, [nombre]!"
Ejercicio 2
Crear una funcion llamada suma que acepte 2 números y devuelva la suma de estos.
Ejericio 3
Crear una funcion llamada esPar que acepte un número y retorne true si el número es par o false si es impar.
Ejericio 4
Crear una funcion llamada mayorDeTres que reciba 3 números y retorne el mayor de ellos.
Ejericio 5
Crear una función llamada celsiusAFahrenheit que acepte una temperatura en grados Celsius y la convierta a Fahrenheit utilizando la fórmula: F = (C * 9/5) + 32.
