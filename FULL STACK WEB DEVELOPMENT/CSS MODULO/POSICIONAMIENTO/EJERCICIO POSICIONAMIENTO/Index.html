¿Qué es el posicionamiento en CSS?

El posicionamiento en CSS permite controlar cómo se ubican los elementos dentro del documento. Existen diferentes tipos: static, relative, absolute, fixed y sticky.

Tipos de posicionamiento

  • Static: Valor por defecto, sigue el flujo normal.
  • Relative: Se posiciona respecto a su posición original.
  • Absolute: Se posiciona respecto al contenedor posicionado más cercano.
  • Fixed: Se fija respecto a la ventana del navegador.
  • Sticky: Actúa como relative hasta cierto punto y luego como fixed.

El posicionamiento es clave para diseñar layouts modernos y controlar la disposición visual sin romper el flujo del documento.
