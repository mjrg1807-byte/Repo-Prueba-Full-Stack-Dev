Guía rápida de CSS Grid

Explicación sencilla para entender CSS Grid desde cero

¿Qué es CSS Grid?

CSS Grid es un sistema que permite organizar elementos en filas y columnas. Es como crear una tabla invisible donde colocas los elementos de una página web.

Idea clave

Grid divide la pantalla en bloques (como una cuadrícula) y te permite decidir dónde colocar cada elemento y cuánto espacio ocupa.

1. Activar Grid

.contenedor {
  display: grid;
}

Convierte un contenedor en un sistema Grid.

2. Crear columnas

.contenedor {
  display: grid;
  grid-template-columns: 1fr 1fr;
}

Crea 2 columnas iguales. "fr" significa fracción del espacio disponible.

Ejemplo de distribución

grid-template-columns: 1fr 2fr;

La segunda columna ocupa el doble de espacio que la primera.

3. Colocar elementos

.item {
  grid-column: 1 / 3;
}

El elemento ocupa desde la columna 1 hasta la 3 (se expande).

Propiedades importantes

Contenedor (padre)

  • display: grid
  • grid-template-columns
  • grid-template-rows
  • gap

Elementos (hijos)

  • grid-column
  • grid-row

Diferencia entre Flexbox y Grid

  • Flexbox → una dirección (fila o columna)
  • Grid → dos dimensiones (filas y columnas)
  • Flexbox → más simple
  • Grid → más potente para layouts

¿Cuándo usar Grid?

  • Diseño de páginas completas
  • Layouts con sidebar
  • Galerías
  • Dashboards

Consejo clave

Siempre piensa:

  • ¿Cuántas columnas necesito?
  • ¿Cuántas filas?
  • ¿Qué elemento ocupa más espacio?
