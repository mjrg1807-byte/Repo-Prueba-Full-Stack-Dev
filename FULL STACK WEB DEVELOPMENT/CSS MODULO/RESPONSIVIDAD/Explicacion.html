Responsividad en CSS

La responsividad consiste en hacer que una página web se adapte correctamente a diferentes tamaños de pantalla, como móviles, tablets y ordenadores.

El objetivo es crear una única web que funcione bien en cualquier dispositivo.

Meta viewport

Es una etiqueta obligatoria en HTML para que la web se adapte correctamente en dispositivos móviles. 

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Unidades flexibles

Permiten que los elementos se adapten al tamaño de la pantalla.

  • %: relativo al contenedor padre
  • vw: ancho de la pantalla
  • vh: alto de la pantalla
  • rem: tamaño relativo al root
  • em: tamaño relativo al padre

Diseño flexible

Se utiliza Flexbox o Grid para crear estructuras que se adapten automáticamente. 

.contenedor {
  display: flex;
  flex-wrap: wrap;
}

Media Queries

Permiten aplicar estilos diferentes según el tamaño de pantalla. 

@media (max-width: 768px) {
  body {
    background-color: lightblue;
  }
}

Breakpoints

  • Móvil: menos de 768px
  • Tablet: entre 768px y 1024px
  • Desktop: más de 1024px

Ejemplo práctico

Adaptar un layout de escritorio a móvil: 

@media (max-width: 768px) {
  .contenedor {
    flex-direction: column;
  }

  aside {
    width: 100%;
    margin-left: 0;
  }
}

Imágenes responsive

img {
  max-width: 100%;
  height: auto;
}

Errores comunes

  • Usar solo px
  • No usar media queries
  • No probar en móvil
  • Diseños rígidos

Consejo clave

Diseña primero para móvil (Mobile First) y luego adapta a pantallas más grandes. 

@media (min-width: 768px) {
  /* estilos para pantallas grandes */
}
