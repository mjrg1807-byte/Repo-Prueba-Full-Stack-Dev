Transiciones, Transformaciones y Animaciones en CSS

Estas propiedades permiten cambiar el estado de los elementos en el tiempo, creando efectos visuales dinámicos y mejorando la experiencia de usuario.

Idea general

  • Transition: cambio suave entre dos estados
  • Transform: modifica forma o posición
  • Animation: animaciones complejas con varios pasos

Transitions

Permiten suavizar los cambios entre un estado inicial y uno final. Normalmente se usan con eventos como :hover. 

.caja {
  background: blue;
  transition: 0.5s;
}

.caja:hover {
  background: red;
}

Sintaxis

transition: propiedad duración tipo delay;

Tipos de transición

  • ease (suave)
  • linear (constante)
  • ease-in (empieza lento)
  • ease-out (termina lento)

Transform

Permite modificar visualmente un elemento sin afectar el flujo del documento.

Tipos principales

transform: translateX(100px); /* mover */
transform: scale(1.5);       /* agrandar */
transform: rotate(45deg);    /* rotar */
transform: skew(10deg);      /* inclinar */

Ejemplo con hover

.caja:hover {
  transform: scale(1.2);
}

Transform + Transition

Combinar ambas permite crear animaciones suaves sin necesidad de usar animation. 

.caja {
  transition: 0.3s;
}

.caja:hover {
  transform: scale(1.2);
}

Animations

Permiten crear animaciones más complejas mediante varios estados definidos.

Crear animación

@keyframes mover {
  from {
    transform: translateX(0);
  }
  to {
    transform: translateX(200px);
  }
}

Aplicar animación

.caja {
  animation: mover 2s infinite;
}

Propiedades importantes

  • infinite (repetición infinita)
  • alternate (ida y vuelta)
  • forwards (mantiene estado final)

Diferencias clave

  • Transition → necesita interacción (hover)
  • Animation → funciona automáticamente
  • Transition → simple
  • Animation → compleja

Ejemplo práctico

.boton {
  background: blue;
  transition: 0.3s;
}

.boton:hover {
  transform: scale(1.1);
  background: red;
}

Consejo importante

Es mejor usar transform en lugar de propiedades como top o left, ya que es más eficiente y mejora el rendimiento. 

/* Evitar */
top: 20px;

/* Recomendado */
transform: translateY(20px);
