Unidades flexibles en CSS

Las unidades flexibles permiten que los elementos se adapten a diferentes tamaños de pantalla. Son fundamentales para crear diseños responsive.

¿Por qué no usar solo px?

Los píxeles (px) son unidades fijas, lo que significa que no se adaptan al tamaño de la pantalla. Esto puede provocar que la web se vea mal en dispositivos móviles.

Unidades más utilizadas

  • %: relativo al tamaño del contenedor padre
  • vw: porcentaje del ancho de la pantalla
  • vh: porcentaje del alto de la pantalla
  • rem: relativo al tamaño de fuente del elemento raíz
  • em: relativo al tamaño de fuente del elemento padre

Ejemplos

.contenedor {
  width: 100%;
}

.caja {
  width: 50%;
}

.texto {
  font-size: 1.2rem;
}

.pantalla {
  width: 100vw;
  height: 100vh;
}

Estos valores permiten que los elementos se adapten automáticamente al tamaño de la pantalla.

Buenas prácticas

  • Usar % para layouts
  • Usar rem para textos
  • Usar vw/vh para dimensiones de pantalla
  • Evitar usar solo px en diseños responsive
