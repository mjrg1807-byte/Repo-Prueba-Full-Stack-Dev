CONTENEDORES

Contenedor div

El contenedor div es un elemento genérico que se utiliza para agrupar contenido dentro de una página web.

No tiene significado semántico, es decir, no indica qué tipo de contenido contiene, solo sirve para organizar elementos o aplicar estilos con CSS.

Etiquetas semánticas

Las etiquetas semánticas son contenedores HTML que describen el tipo de contenido que contienen, ayudando al navegador, a los buscadores y a los lectores de pantalla a entender mejor la estructura de la página.

Ejemplos de etiquetas semánticas

  • <header>: cabecera del sitio o sección
  • <nav>: menú de navegación
  • <main>: contenido principal
  • <section>: sección temática
  • <article>: contenido independiente
  • <footer>: pie de página

Diferencia entre div y etiquetas semánticas

Mientras que el div solo agrupa contenido sin aportar significado, las etiquetas semánticas indican claramente la función del contenido que contienen, mejorando la accesibilidad y la organización del código.
