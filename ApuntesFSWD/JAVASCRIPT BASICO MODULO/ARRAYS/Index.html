Arrays en JavaScript

Un array es una estructura de datos que permite almacenar múltiples valores dentro de una sola variable.

¿Qué es un array?

Es una lista de elementos que se pueden recorrer y manipular fácilmente. 

let numeros = [1, 2, 3, 4];
let nombres = ["Mauro", "Juan", "Ana"];

Índices (posiciones)

Cada elemento del array tiene una posición llamada índice. Los índices comienzan en 0. 

let nombres = ["Mauro", "Juan", "Ana"];

console.log(nombres[0]); // Mauro
console.log(nombres[1]); // Juan

Propiedad length

La propiedad length indica la cantidad de elementos del array. 

let nombres = ["Mauro", "Juan", "Ana"];

console.log(nombres.length); // 3

Recorrer un array

Se puede recorrer un array utilizando un bucle for. 

let nombres = ["Mauro", "Juan", "Ana"];

for (let i = 0; i < nombres.length; i++) {
  console.log(nombres[i]);
}

Modificar arrays

Añadir elementos

nombres.push("Luis");

Eliminar el último elemento

nombres.pop();

Modificar un valor

nombres[0] = "Pedro";

Tipos de datos en arrays

Un array puede contener diferentes tipos de datos. 

let mezcla = ["Mauro", 25, true];

Cuándo usar arrays

  • Cuando trabajas con listas de datos
  • Cuando necesitas agrupar información
  • Cuando vas a recorrer múltiples valores

Errores comunes

  • Pensar que los índices empiezan en 1
  • Olvidar usar .length en bucles
  • No entender qué contiene el array

Resumen

  • Array → lista de datos
  • Índice → posición del elemento
  • length → número de elementos
  • for → recorrer el array
