Imágenes en HTML

En HTML, las imágenes se utilizan para mostrar contenido visual dentro de una página web. Se insertan mediante la etiqueta img, que es una etiqueta autocerrada.

Etiqueta img

La etiqueta img se usa para insertar imágenes. No tiene etiqueta de cierre y funciona mediante atributos.

Ejemplo básico de imagen

El atributo src indica la ruta de la imagen, y alt proporciona una descripción alternativa.

Atributos importantes de las imágenes

  • src: ruta donde se encuentra la imagen
  • alt: texto alternativo (obligatorio por accesibilidad)
  • width: ancho de la imagen
  • height: alto de la imagen
  • title: texto que aparece al pasar el ratón

Uso semántico con figure y figcaption

Cuando una imagen tiene un significado propio o necesita una explicación, se recomienda usar las etiquetas <figure> y <figcaption>.

Rutas de imágenes

Ruta relativa

Se usa cuando la imagen está dentro del proyecto.

Ruta absoluta

Se usa cuando la imagen está en internet.

Accesibilidad en imágenes

El atributo alt es fundamental para personas con discapacidad visual y para cuando la imagen no se puede cargar.

Buenas prácticas al usar imágenes

  • Usar siempre el atributo alt
  • No usar imágenes solo para texto
  • Optimizar el tamaño de las imágenes
  • Usar figure cuando la imagen tenga significado
  • No usar HTML para escalar imágenes grandes
